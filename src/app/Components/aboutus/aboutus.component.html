Cplayer App

The CplayerApp is a website is a meant to be an experiment in providing top-quality data and has the ability to get the latest cricket scores is today limited to those with deep pockets. This website consists of 3 sections Favourites,Player Statistics,Recommendation. The PlayerStatistics will display all the information about the player along with the Statistics of the player.The Favourites section will have the data which the user wanted to view the profile of the player regularly.The Recommendation section we most viewed profiled by all the users globally.This project is made to know the complete details of the player.We can search the players based on the Pid, name and fullname also.This project is given by StackRoute made by Akash das, Pranathi, Swetha.
